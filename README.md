**Project Rias**

Simple JDA music and chat reactions discord bot.

Developed using this libs:

* `JDA`
* `lavaplayer`

**Setup:**

1) Clone repo

2) Write your `token` to .env file

3) Start Rias-1.0.jar

Bot working directory example:
* `Rias-1.0.jar`
* `.env`


**Commands:**

* `/emote <emote_type> [mention]`
* `/ping`
* `/play <Song>`
* `/nowplaying`
* `/queue`
* `/repeat`
* `/skip [amount]`
* `/stop`
* `/leave`
